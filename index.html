Explore Maps

What is the purpose of this website?

Hello! My name is Gabrielle, and I am a student in the class Fundamentals of Geographical Informational Systems. The purpose of this website is to showcase three of my favorite maps I have created throughout the course of this class using the program ArcGIS Pro. I will be explaining why I created these maps, as well as the skills I learned throughout the production process.

Exploring Map Projections

For this excercise given in Lab 2, we explored several different map projections: Mercator, Robinson, Goode Homolosine (Land). This map projection shown is known as "Moon 2000 North Pole Stereographic". In order to achieve this, you must go to the coordinate systems tab by right clicking on the map you want changed, and hitting properties. Once in the coordinate systems tab, you are able to browse hundreds of options to display your map in. This skill is important to know because a projection can change the way data is calculated, or displayed.

Hispanic Population Density

Lab 3 focused on the population density of the Hispanic population in the United States. We were given a set of data that included the stats on how many Hispanics live in each state, where we needed to manually calculate the population denisty per square mile in each block. We were looking for the states with the highest Hispanic population, the lowest, and which one has a higher Hispanic population than the least populous state: Wyoming. I had made this choropleth map to display the distribution of Hispanics among each state. The symbology tab allows you to make different styles of maps depending on the circumstance. This map was created using the Graduated Colors option. The title and legend were added after creating a new layer and adding the newly made choropleth map.

Vacancy across Miami

This last map from the second Mastery assignment represents the percentage of vacant homes across Miami-Dade county in Florida. Along with this map, I created one representing occupied homes and population density of each block in the county. All three of these fields needed to manually be calculated with the dataset provided. Since this map is a choropleth map, it was created and presented in the same way as the Hispanic Population Denisty map above. This map especially stood out to me due to the highest percentage of vacant homes in one census block: 99.12%.
